    Variable

    Un variable conceptualmente es un contenedor etiquetado el cual almacena un valor 1. Para una definición más detallada puedes pulsar aquí.

    Ejemplo:

    >>> nombre = "John"
>>> edad = 18

    La variable nombre almacena el valor "John" (cadena de texto) y la variable edad almacena el valor 18 (entero).

    Para imprimir los valores de esas variables usamos la función print:

    >>> print(nombre)
John
>>> print(edad)
18

    Comentarios

    Un comentario es una instrucción no ejecutable, un texto que se usa como ayuda para detallar información relevante y tiene por finalidad hacer más comprensible el código.

    Los comentarios inician con # seguido de un espacio en blanco.

    # Comentario de una línea

    Para bloques de comentarios ademas se respeta el nivel de indentación.

    # Comentario
# de varias
# líneas

    NOTA: El uso de comillas simples (') o comillas dobles (") se limitan a cadenas multilínea y docstrings (cadenas de documentación prefiriendo comillas dobles).

    Identificador

    Un identificador es un nombre que puede darse a una variable, función, clase, módulo u otro objeto (en Python todo es un objeto).

    Los identificadores empiezan con una letra (A-Z, a-z) o con guión bajo (_), seguido de más letras, guiones bajos o dígitos.

    >>> nombre = "John"
>>> apellido = "Doe"
>>> edad = 18
>>> print(f"{nombre} {apellido} tiene {edad} años")
John Doe tiene 18 años

    IMPORTANTE: La guía de estilo de Python, conocida como PEP8 2, contiene las convenciones de nomemclatura que enumeran estándares sugeridos para nombres de diferente tipos de objetos. PEP8 incluye las siguientes recomendaciones:

    • snake_case debe ser usado para nombrar funciones y variables.
    • CamelCase debe ser usado para classes.

    Los caracteres de puntuación como @, $ y ! no estan permitidos como identificadores.

    >>> @nombre = "John"
  File "<stdin>", line 1
    @nombre = "John"
            ^
SyntaxError: invalid syntax

    Las mayúsculas y minúsculas son distintivas, jugadores no es igual a JUGADORES, de la misma forma Jugadores no es igual a jugadores.

    >>> jugadores = 2
>>> JUGADORES = 11
>>> print(jugadores == JUGADORES)
False

    Palabras reservadas

    Es un conjunto de palabras claves de uso exclusivo del lenguaje, por lo tanto no deben ser utilizadas como identificadores:

pass    raise   return  try       while     with    yield

    TIP: Se puede ver la lista de palabras reservadas tipeando lo siguiente en el interprete de Python

    >>> help("keywords")

    Definiendo variables

    Para definir una variable en Python, simplemente asignamos un valor a un identificador. Por ejemplo:

    >>> nota = 20
>>> print(nota)
20
