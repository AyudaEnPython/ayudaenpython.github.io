    Cómo realizar buenas preguntas

    Tabla de Contenidos

    • Describe el problema y el contexto
    • Comparte tu código

    Describe el problema y el contexto

    Un simple “alguien me puede ayudar?” no basta. Se más específico, brinda más detalles.

    Comparte tu código

    Evita compartir imágenes… porque:

    • Las imágenes muestran un número limitado de líneas de código.
    • Si las imágenes que compartes no tiene buena resolución, serán ilegibles.
    • Si el código de la imágen está escrito en una fuente pequeña, será difícil de leer.
    • No se puede copiar el código de una imagen para reproducir el problema.
    • Las imágenes no permiten la edición colaborativa del código.

    Si estas pidiendo ayuda para resolver tu problema, trata de facilitar las cosas; en lugar de usar imágenes, comparte en formato de texto usando servicios como GitHub, PasteBin, Replit, y similares.

    No elimines tu pregunta/consulta

    Tu pregunta/consulta puede ser de utilidad a otra persona, por lo tanto no la elimines si se solucionó o nadie pudo ayudarte.
