    Encabezados

    Para crear un encabezado, agregar entre uno y seis símbolos # antes del encabezado del texto. El número de # que se utilice determinará el nivel jerárquico y el tamaño tipográfico del encabezado.

    Sintaxis:

    # Encabezado de primer nivel
## Encabezado de segundo nivel
### Encabezado de tercer nivel
#### Encabezado de cuarto nivel
##### Encabezado de quinto nivel
###### Encabezado de sexto nivel

    Resultado:

    Encabezado de primer nivel

    Encabezado de segundo nivel

    Encabezado de tercer nivel

    Encabezado de cuarto nivel

    Encabezado de quinto nivel
    Encabezado de sexto nivel


    También se puede utilizar la siguiente sintaxis para los encabezados de primer y segundo nivel:

    Encabezado de primer nivel
==========================

Encabezdo de segundo nivel
--------------------------

    Resultado:

    Encabezado de primer nivel

    Encabezado de segundo nivel

    Estilos de texto

    Se puede indicar el énfasis con texto en negrita, cursiva, tachado, o de subíndice o superíndice.

    Descripción Sintaxis Ejemplo Resultado
    Negrita ** ** o __ __ **Texto en negrita** Texto en negrita
    Cursiva * * o _ _ _Texto en cursiva_ Texto en cursiva
    Tachado ~~ ~~ ~~Texto tachado~~ Texto tachado
    Cursiva en negrita y anidada ** ** y _ _ **Este texto es _muy_ importante** Este texto es muy importante
    Todo en negrita y cursiva *** *** **Todo este texto es importante** Todo este texto es importante
    Subíndice <sub> </sub> Esto es un <sub>subíndice</sub> Esto es un subíndice
    Superíndice <sup> </sup> Esto es un <sup>superíndice</sup> Esto es un superíndice

    Listas

    Se puede crear una lista sin ordenar usando -, * o + antes de una o más líneas de texto.

    Sintaxis:

    - Python
* Rust
+ JavaScript

    Resultado:

    • Python
    • Rust
    • JavaScript


    Para ordenar una lista, anteceder cada línea con un número.

    Sintaxis:

    1. Html
2. CSS
3. JavaScript

    Resultado:

    1. Python
    2. Rust
    3. JavaScript


    Para crear una lista de tareas se utiliza la siguiente sintaxis:

    - [x] Documentar la nueva funcionalidad.
- [ ] Refactorizar.
- [ ] Enviar documentos.

    Resultado:

    • Documentar la nueva funcionalidad.
    • Refactorizar.
    • Enviar documentos.


    Notas al pie

    Se pueden agregar notas al pie para el contenido usando la siguiente sintaxis:

    Python es un lenguaje interpretado, dinámico y multiplataforma[^1].

[^1]: https://es.wikipedia.org/wiki/Python

    Resultado:

    Python es un lenguaje interpretado, dinámico y multiplaforma.1

    1. https://es.wikipedia.org/wiki/Python
